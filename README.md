# UrbanBites Cafe Website

A clean, responsive multi-page cafe website built with HTML, CSS, and JavaScript.

## Preview

![UrbanBites Cafe UI Preview](./preview.png)

> Preview generated from the current homepage UI (`index.html`).

## Pages

- Home: `index.html`
- About: `about.html`
- Menu: `menu.html`
- Gallery: `gallery.html`
- Contact: `contact.html`

## Project Structure

- `css/style.css` - Main styling
- `js/main.js` - Main JavaScript
- `robots.txt` - Search engine crawl rules
- `sitemap.xml` - Site map for indexing

## Run Locally

1. Open `index.html` in your browser.
2. Or use any local server extension/tool to run the site.

## Notes

- This is a static frontend project.
- No build step is required.
